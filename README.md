# Halo 文章加密插件

> 🔐 支持区块加密、全文加密、分类加密，TOTP 动态密码，加密过期自动公开

## ✨ 功能特性

### 🔒 多种加密方式

| 方式 | 说明 |
|-----|------|
| **区块加密** | 加密文章部分内容 |
| **全文加密** | 一键加密整篇文章 |
| **分类加密** | 按分类批量设置密码 |

### 🔑 灵活的认证方式

| 方式 | 说明 |
|-----|------|
| **固定密码** | 传统密码保护 |
| **TOTP 动态密码** | 可配置有效期（30秒/1小时/1天/1周/1月） |
| **万能密钥** | 管理员备用密码 |

### ⏰ 加密过期

设置加密期限，到期后内容自动变为公开：
- 7天后公开
- 30天后公开
- 自定义日期

### 🎨 UI 特性

- ✅ 解锁动画（浮动图标/发光效果）
- ✅ 深色模式支持
- ✅ 毛玻璃效果
- ✅ 移动端适配

### 🛡️ 安全功能

- ✅ 密码错误锁定（可配置次数和时间）
- ✅ BCrypt 密码加密
- ✅ XSS 防护
- ✅ 会话记忆（刷新保持解锁）

## 📥 安装

### 方式一：应用市场
1. 进入 Halo 后台 → 插件
2. 搜索「文章加密」
3. 点击安装

### 方式二：手动安装
1. 下载 [Releases](https://github.com/jcwang99/encryption_halo/releases) 中的 jar 文件
2. 上传到 Halo 后台 → 插件

---

## 🚀 使用方法

### 区块加密

在编辑器中使用工具栏按钮：

```
[encrypt password="your_password"]
这里是加密内容...
[/encrypt]
```

### 全文加密

点击编辑器工具栏的「加密」按钮 → 选择「全文加密」

### 加密过期

```
[encrypt password="vip" expires="2025-12-31"]
限时 VIP 内容，2025年12月31日后自动公开
[/encrypt]
```

### TOTP 动态密码

1. 进入 工具 → 动态密码
2. 点击「生成密钥并启用」按钮（自动保存）
3. 复制显示的密码分享给用户
